## КРИТИЧЕСКИЕ ДОПУЩЕНИЯ (явные)

### A1: Калибровочная группа — компактная связная Lie группа

**Принимается как физический вход.**

Обоснование:
- **Компактность**: Унитарность квантовой механики (∑|ψ|² = 1 сохраняется)
- **Связность**: Симметрии непрерывны (можно плавно деформировать)
- **Lie**: Дифференцируемость, генераторы с коммутаторами

**DD не выводит это** — это предусловие.

### A2: Физическая интерпретация "цвета"

**Принимается**: Omega = внутренние степени свободы кварков.

Обоснование:
- Глубоконеупругое рассеяние: 3 валентных кварка в протоне
- R-ratio: R = ∑Q² × N_c = ... требует N_c = 3
- Confinement: только цвет-синглеты наблюдаемы

**DD формализует структуру**, физика даёт интерпретацию.

### A3: Distinction = нетривиальный автоморфизм

**Принимается**: Δ ≠ ∅ ↔ существует f: A → A, f ≠ id.

Обоснование:
- Математически: различие = возможность сравнения
- Физически: измерение = сравнение с эталоном

**DD не объясняет "почему есть различие"** — это axiom.
